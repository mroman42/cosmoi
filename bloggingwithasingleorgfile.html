Blogging with a single org-file

This blog is generated by a single org-file which can be found here. The file contains some org-babel elisp blocks that export the HTML files, create the index and populate the RSS feed. This is possible using tags that signal whether any specific header is a post, properties that store the name and the html direction of the articles, and the org-map-entries function, which maps a function over all headers.
