Discokitty, an implementation

As a part of the course on Distributional Models of Meaning by Coecke and Marsden, I wrote an educational implementation of the DisCoCat framework (as described in "Mathematical Foundations for a Compositional Distributional Model of Meaning").

Discokitty is written on Haskell and the code can be found on GitHub.
