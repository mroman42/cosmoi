Yoneda lemma

For any covariant K : D → Sets and r ∈ D, there is a bijection 

   y  :  Nat(D(r,-), K) ≅ Kr

sending any natural transformation α : D(r,-) => K to its image on the identity, αᵣ(idᵣ). If we call よ : C -> Cat(Cᵒᵖ, Sets) to the Yoneda embedding that sends each object to its representable presheaf, we can also write 

   Nat(よ(r), K) ≅ K r

The formulation in terms of ends and coends is often called the Ninja Yoneda lemma (see Loregian's This is the (co)end, my only (co)friend). It can be seen as a sort of Dirac's rule for coend calculus.

For any functor K : Cᵒᵖ -> Sets (any presheaf), we have 

  K  ≅  ∀ c ∈ C .  C(c,_) -> K c
  K  ≅  ∃ c ∈ C .  K c × C(_,c)

For any functor H : C -> Sets (any copresheaf), we have 

  H  ≅  ∀ c ∈ C .  C(_,c) -> H c 
  H  ≅  ∃ c ∈ C .  H c × C(c,_)
